Mood-Based Music Recommendation System

This project predicts user mood from physiological signals and recommends music.

Files:

train_model.py  
Trains classification model.

app.py  
Streamlit application for mood prediction and music recommendation.

Pipeline:

CSV → ML Model → Streamlit

How to Run:

python train_model.py  
python -m streamlit run app.py  

Output:

Predicted Mood  
Recommended Music Genre
